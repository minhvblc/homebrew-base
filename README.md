# homebrew-base

Homebrew tap for the internal `base` scaffolding CLI.

## Install

If this tap is public or readable through standard GitHub access:

```bash
brew install minhvblc/homebrew-base/base
```

If this tap is private, do a one-time tap with the Git URL, then install:

```bash
brew tap minhvblc/homebrew-base git@github.com:minhvblc/homebrew-base.git
brew install base
```

## Upgrade

```bash
brew update
brew upgrade base
```

## First use

```bash
base new
```
